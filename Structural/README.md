# 结构型模式

这些设计模式关注**类和对象的组合**。
继承的概念被用来组合接口和定义组合对象获得新功能的方式。

## 适配器模式

* 简介

作为两个**不兼容**的接口之间的**桥梁**。
它结合了两个独立接口的功能。

* 示例

> 场景：书本与电子书的翻页方式。

## 桥接模式

* 简介

桥接是用于把**抽象化与实现化解耦**，使得二者可以独立变化。
这种模式涉及到一个作为桥接的接口，使得实体类的功能独立于接口实现类。这两种类型的类可被结构化改变而互不影响。

* 示例

> 场景：图片和视频的存储服务,与实现存储的功能接口
